Animation

  • Transitions get use from point A to point B

  • But sometimes we need some more complicated behavior

  • CSS3 Animations let us define keyframes and then how to apply them

How?

/* Define the keyframe */
@-webkit-keyframes bounce {
   0% { -webkit-transform: translate(0,0); }
   50%  { -webkit-transform: translate(225px,350px); }
   100% {  -webkit-transform: translate(450px, 0); }
}

/* This is the element that we apply the animation to. */
.ball {
   -webkit-animation-name: bounce;
   -webkit-animation-duration: 1s;
   -webkit-animation-timing-function: ease; /* default ease, (also: linear) */
   -webkit-animation-delay: 1s;             /* default 0 */
   -webkit-animation-iteration-count: 3;    /* default 1 (also: infinite) */
   -webkit-animation-direction: alternate;  /* default normal */
}
/* or */
.ball {
  -webkit-animation: bounce 1s ease 1s 3 alternate;
}
