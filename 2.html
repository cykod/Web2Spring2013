More tricks

  • Use :before and :after to limit number of HTML tags you need (need content)

         .block:after {
        content:"";
        width:30px;
        height:30px;
        background-color:red;
     }

  • Use absolute positioning to move stuff around

  • Use the longer form of border-radius to pick individual sides to radius:

         .half-moon {
        width:100px;
        height:100px;
        border-radius: 0 0 100px 100px; 
     }

  • Use borders to create lines instead of full shapes

Vendor Prefixes

  • What are they?

  • They help the web develop at faster rate

  • Let developers use cool stuff, but know: Here be dragons

  • Won't work across all browsers, won't keep working the way they used to.

Prefixed transform

  • Here's an example:

    
<style>
.box_rotate {
 -webkit-transform: rotate(7.5deg);  /* Chrome, Safari 3.1+ */ 
    -moz-transform: rotate(7.5deg);  /* Firefox 3.5-15 */ 
     -ms-transform: rotate(7.5deg);  /* IE 9  */
      -o-transform: rotate(7.5deg);  /* Opera 10.50-12.00  */
         transform: rotate(7.5deg);  /* FF16+, IE10+, O12.50+  */
}
</style>

  • There are a number of JS workarounds (Prefix Free)

  • ..But they add complication and have limitations..

  • In this class ONLY: you can just use the -webkit- versions

  • In the real world (or if you want your work to live past this class): use all prefixes
