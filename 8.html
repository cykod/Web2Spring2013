Objects

  • A way to store data and methods together.

  • Array was an example of a built in object

  • Can create with JSON notation:

    var obj = { a: 1, b: 2 };

  • Access the same as arrays - with square brackets.

    obj['a'] // 1 
obj['b'] // 2

  • Can also access with the dot operator

    obj.a // 1
obj.b // 2

  • Can also have functions as properties

  • Those functions can use the this property when called

Same Domain Origin policy

  • You can't just pull data form anywhere

  • It has to be on the same domain

  • And it doesn't work from file:// URLS

JSONp (JSON with padding)

  • Allows us to pull data with script tags

  • Not affected by the same origin policy

  • Must be supported via the API

  • Implementation is API specific, but often uses callback=?
